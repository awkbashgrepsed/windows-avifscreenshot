# windows-avifscreenshot
take avif screenshots with the print key on windows

* Runs in the system tray
* Global Print Screen hotkey
* Captures whichever window is currently focused
* Works across multiple monitors
* Saves automatically to Pictures
* Uses .avif for smaller files
* Generates its tray icon internally

# Usage
install requirements
```python
python -m pip install -r requirements.txt
```
run
```python
python main.py
```
